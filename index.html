Introdução

DNS, acrônimo para Domain Name System, é o sistema responsável pela tradução de nomes de domínio em endereços IP, que por sua vez são interpretados pelas interfaces de rede. Na internet um cliente web faz solicitações DNS para descobrir para qual endereço IP devem ser enviados os datagramas. As requisições e respostas DNS utilizam como protocolo de transporte o UDP (User Datagram Protocol).

Os servidores DNS possuem bancos de dados da forma (type, name, value, TTL), onde type define o tipo de consulta que será feita ao servidor, name corresponde ao domínio, value ao respectivo endereço IP e TTL o tempo que o registro será mantido no cache do servidor.(Kurose, 2016) Nenhum servidor possui registros de todos os domínios na Internet, pois o sistema funciona de forma hierarquica e distribuída, de modo que seja escalável nas infraestruturas de redes.
