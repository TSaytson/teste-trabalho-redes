DNS, acrônimo para Domain Name System, é o sistema responsável pela tradução de nomes de domínio em endereços IP, que são interpretados pelas interfaces de rede, por serem de tamanho fixo (4 bytes), diferentemente dos nomes de domínio. A integridade e a disponibilidade do DNS são essenciais para o funcionamento e para a segurança da Internet. Como o DNS é vital para uma vasta gama de aplicações, desde um roteamento seguro à distribuição de certificados, ataques a essa estrutura podem ser lucrativos para quem os pratica. Na internet um cliente web faz solicitações DNS para descobrir para qual endereço IP devem ser enviados os datagramas. O DNS pertence à camada de aplicação e suas requisições e respostas utilizam UDP (User Datagram Protocol) como protocolo de transporte.
